🔧 修复 Mixed Content 错误

❌ 问题描述

错误：blocked:mixed-content

原因：您的网站使用HTTPS，但API调用可能使用了HTTP协议，导致浏览器阻止了不安全的请求。

✅ 解决方案

需要确保所有API调用都使用HTTPS协议。

🔍 问题诊断

1. 检查当前API地址

当前API地址: https://1300931050-izxeco6na5.ap-guangzhou.tencentscf.com

2. 测试API连接

3. 检查CORS配置

🛠️ 修复步骤

步骤1：确认API地址使用HTTPS

✅ 正确: https://1300931050-izxeco6na5.ap-guangzhou.tencentscf.com ❌ 错误: http://1300931050-izxeco6na5.ap-guangzhou.tencentscf.com

步骤2：检查前端代码中的API调用

确保所有fetch请求都使用HTTPS：

// 正确的API调用 const response = await fetch('https://1300931050-izxeco6na5.ap-guangzhou.tencentscf.com/ocr', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify(data) });

步骤3：重新生成静态文件

如果前端代码中有HTTP调用，需要重新生成静态文件：

1. 检查 src/components/ 中的API调用 2. 确保所有API地址都使用HTTPS 3. 重新运行 export-static.bat 4. 重新上传out目录到COS
